# 黑白棋 (Othello/Reversi)

## 如何运行

### 方法一：你的电脑已有 Python（推荐）

1. 确保安装 Python 3.7 或更高版本
   - 下载: https://www.python.org/downloads/
   - 安装时勾选 **"Add Python to PATH"**
2. 解压本压缩包
3. 双击 `run.bat`

### 方法二：没有 Python

1. 下载 Python 安装包（约 25MB）:
   https://www.python.org/ftp/python/3.11.9/python-3.11.9-amd64.exe
2. 运行安装程序，务必勾选 **"Add Python to PATH"**
3. 解压本压缩包，双击 `run.bat`

---

## 游戏说明

- **先手/后手**：开局弹窗选择
- **设置初始棋盘**：点击菜单按钮进入编辑模式
- **左键**落子，**绿色圆点**表示合法位置
- AI 棋力较强，请认真对待 :)
